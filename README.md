# DSA Task Scheduler (C++)

This project implements a simple task scheduler using C++ and data structures.
Tasks are scheduled based on priority and execution order.

## Features
- Add tasks with priority
- Execute tasks based on priority
- Remove completed tasks

## Concepts Used
- C++
- Data Structures
- STL (priority_queue)
- Basic OOPS

## How to Run
1. Compile the program using a C++ compiler
2. Run the executable
3. Follow the menu options
